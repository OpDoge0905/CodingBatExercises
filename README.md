## 📘 專案介紹

此儲存庫用於：

- 系統化練習 **Java 基礎語法、邏輯判斷、迴圈、陣列、字串處理**
- 訓練 **演算法思維與解題能力**
- 作為我學習進度與程式能力成長的 **紀錄與展示**
- 方便教授、助教或未來面試官快速了解我的 **程式風格與實力**

所有題目來自 **CodingBat Java**，並依照題目類型清楚分類。
